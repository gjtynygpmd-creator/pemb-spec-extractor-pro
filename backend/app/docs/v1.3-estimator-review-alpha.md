# v1.3 Estimator Review Alpha

This release turns extracted results into an estimator review workflow.

## Included
- Card-based extracted field review
- Editable field values and review statuses
- Accept, conflict, review, and not-applicable states
- Source document, sheet, page, confidence, and excerpt display
- Required-field missing-information checklist
- Conflict center
- Review-completion indicator
- Manual entry for missing fields
- Category and status filters

## Deliberately deferred
- Embedded PDF viewer and signed source-page links
- OCR for image-only pages
- AI document Q&A
- Zoho export
- Proposal generation

These are planned after the review workflow is validated on real estimates.
